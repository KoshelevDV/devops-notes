## 📝 Формат заметок

Каждая заметка содержит:
- 🎯 **Проблема** — что за вопрос или ситуация
- ✅ **Решение** — шаги, команды, примеры
- 📎 **Документация** — ссылки на официальные доки
- 🔗 **Источники** — статьи, Reddit, форумы
- 💡 **Подводные камни** — что может пойти не так
